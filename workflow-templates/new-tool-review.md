# New Tool Review Workflow
Quick process for reviewing new tool submissions to maintain quality and consistency.

## Quick Steps (15-30 min total)
1. **First Look (5 min)**
   - [ ] Uses standard template
   - [ ] Category & difficulty make sense
   - [ ] All sections present
   - [ ] Links work
   - [ ] Sources cited properly

2. **Quality Check (10-20 min)**
   - [ ] Content is clear and actionable
   - [ ] Integration section links correctly
   - [ ] Examples are practical
   - [ ] Collapsible sections used appropriately
   - [ ] Passes accessibility check

3. **Decision (5 min)**
   - [ ] Approve or request changes
   - [ ] Add relevant labels
   - [ ] Post standard response (see below)
